About

Welcome to my RPG Character Creator for D&D 5e

This project works to allow you to create and manage D&D Characters used within the 5e cmapaigns.

You are able to create a new character by navigating to the Create Character button in the navbar menu above, edit them from the Home screen by clicking the Edit pencil, and delete them, but only once they have died in-game!

If you find you are having trouble deleting a character, please edit them and set the Death toggler to on, this will then allow you to delete them. All dead characters will have an icon next to their name in the Home screen. The remove icon should also be shown if this is the case.

All information presented is stored in a cloud based database known as MongoDB, if any problems appear that aren't resolvable within this website, please feel free to contact the developer of this project at bendoo12345@gmail.com

I will gladly answer any questions sent through to this email also, so please feel free to contact me about design requests or for any help on getting started with this website.
